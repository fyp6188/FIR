Privacy Policy

This privacy policy describes how this Chrome extension handles user information. The extension is designed to operate entirely within the user’s browser and does not collect or transmit personal data.

Data Collection

This extension does not collect, store, process, or transmit any personal, sensitive, or identifiable user data. No FIR data, CNIC information, credentials, or browsing activity is accessed or sent outside the browser.

Data Usage

All automation and functionality are performed locally within the browser on supported Punjab Police FIR System pages. The extension does not use analytics, tracking technologies, or third-party services.

Data Sharing

No user data is sold, shared, or transferred to third parties for any purpose.

Remote Code

This extension does not load or execute remote code. All scripts and resources are bundled within the extension package.

Changes

This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.

If you have any questions about this privacy policy, you may contact the developer.
